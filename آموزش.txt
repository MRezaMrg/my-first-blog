git add -A
دستور بالا فایل های جدید را به گیت اضافه میکند

git commit -m "first commit"
دستور بالا فایل های اضافه شده را تایید یا کامیت میکند

git push origin
بعد از وارد کردن دستور بالا فایل ها شروع به آپلود در گیت شما میشوند

$ git status
$ git add .
$ git status
$ git commit -m "Added view and template for detailed blog post as well as CSS for the site."
$ git push
